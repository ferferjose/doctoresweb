Doctors

This is the main menu of the application where we can choose to access the FTP, e-mail, or the manual.

FTP

This is the main screen of the FTP section where we can manage the files.

Upload file

To upload a file, we need to go to the "Upload File" section and then select it.

FTP

Now, as we can see, in the directory /ialonsocalzado/pacientes, there is the file that we just uploaded.

Download

Now, let's download a file. To do this, we should select the file we want to download and click on the "Download" section. Then, we can choose a location for the file.

Delete

To delete a file, we need to select it and in the left section, click on "Delete File."

Rename

To rename a file, select it and click on the "Rename" section in the left panel.

Create a folder

In the left panel, you can see an icon with a folder and a + sign. This is how you can create a new folder and enter a name for it.

Delete a folder

If you click on the icon to the left of a folder with a -, it will prompt you for the folder name, and you can delete it.

If the folder contains files, you will receive an error message.

Email

This is the main email inbox where you can view received emails, read, and send emails.

Send Email

Here, you can send emails, go back to the inbox, add an attachment, write a recipient, and send it.

Check Email

If you enter from the inbox into a received email, you will be able to see the content of the email.
