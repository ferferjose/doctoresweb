Select Menu

This is the main menu of the application where we can decide to access the FTP, email, or the manual.

FTP

This is the main screen of the FTP section where we can manage the files.

Upload File

To upload a file, we have to click on the "Upload File" section and then select it.

FTP

Now, as we can see, in the directory /ialonsocalzado/pacientes, there is the file that we just uploaded.

Download

Now, let's download a file. To do this, we need to select the file we want to download and click on the "Download" section. Then, we can choose a location for the file.

Delete

To delete a file, we need to select it, and in the left section, click on "Delete File."

Rename

To rename a file, select it and click on the "Rename" section in the left panel.

Create a folder

In the left panel, you can see an icon with a folder and a + sign. This is how you can create a new folder and enter a name for it.

Delete a folder

If you click on the icon to the left of a folder with a -, it will prompt you for a folder name, and you can delete it.

If the folder contains files, you will receive an error message.
